# Title:
Interpreting and Iterating on Multiple Regression Models
# Overview:
This exercise has an initial regression models that has several flaws that need to be interpreted, in order to minimize it's effects through iterations of the models that are meant to help correct the flaw.
# Methods:
By analyzing the initial Residuals vs Fitted Plot, Q-Q Residual Plot, Scale-Location Plot and Residuals vs Leverage Plots, we can determine several issues with the current regression model. This is further aided by Residual vs Independent Plots that allows us to determine the effects of different independent variables has on the independent of the model. Furthermore, an plot of Square Root of Absolute Standardized Residuals vs Independent Variable are used to analyze the effects of independent variables on the homoskedasticity of the regression model. An command from the R Studio package car, VIF is further used to guage the effects of independent variables on the multicollinearity of the regression model. Through trial and error, using Stepwise Regression, we can manage to salvage the regression model through the best of our ability.
